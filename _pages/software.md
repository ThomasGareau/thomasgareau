## Research Software

### qesR

**qesR** is an R package that provides easy access to Quebec Election Study datasets,
including harmonized cross-study files for longitudinal analysis.

<span class="pub-links">
  <a href="https://github.com/ThomasGareau/qesR" target="_blank">[GitHub]</a> |
  <a href="https://thomasgareau.github.io/qesR/" target="_blank">[Documentation]</a>
</span>

Key features:

- Download individual QES waves with survey-code calls
- Retrieve study codebooks and variable labels
- Build a harmonized merged dataset across study years
- Access bilingual examples for sovereignty and vote-choice analyses
