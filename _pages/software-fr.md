## Logiciels de recherche

### qesR

**qesR** est un package R qui permet d'accéder facilement aux jeux de données de l'Étude électorale québécoise,
y compris un fichier harmonisé entre études pour les analyses longitudinales.

<span class="pub-links">
  <a href="https://github.com/ThomasGareau/qesR" target="_blank">[GitHub]</a> |
  <a href="https://thomasgareau.github.io/qesR/" target="_blank">[Documentation]</a>
</span>

Fonctionnalités principales :

- Télécharger chaque vague de l'EEQ via un code d'étude
- Récupérer les codebooks et les étiquettes de variables
- Construire un fichier fusionné harmonisé entre années
- Accéder à des exemples bilingues sur la souveraineté et le choix de vote
